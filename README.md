# AI Empire - 大模型公司模拟经营游戏

一款增量游戏（Idle Game），模拟经营一家AI大模型公司。

## 🎮 游戏玩法

### 核心循环
1. **手动训练** - 点击中间的训练按钮获取算力
2. **购买硬件** - 使用资金购买CPU、GPU等硬件自动产出算力
3. **雇佣员工** - 招募工程师提升效率
4. **升级解锁** - 达到里程碑解锁更高级的设施

### 资源系统
- 💰 **资金** - 购买建筑和升级的货币
- ⚡ **算力** - 核心资源，自动转换为收入
- 👥 **用户** - 随着收入增长而增加
- 📊 **数据** - 基于算力自动累积

### 建筑系统
- 🖥️ **CPU集群** - 基础算力生产，成本低
- 🎮 **GPU服务器** - 中级算力生产
- ⚙️ **TPU阵列** - 高级算力生产（需解锁）
- 👨‍💻 **工程师** - 提升算力效率

### 里程碑系统
- 每购买25个同类建筑，产出×2
- 升级等级提升点击倍率
- 成就系统记录游戏进度

## 🚀 如何开始

1. 打开 `index.html` 即可开始游戏
2. 点击中间的"手动训练"按钮开始获取算力
3. 当资金足够时购买第一个CPU集群
4. 享受自动产出的乐趣！

## ⌨️ 快捷键

- `Space` - 手动训练
- `Q/W/E/R` - 快速购买对应建筑（开发中）
- `1-6` - 切换标签页（开发中）

## 💾 存档系统

- 游戏每30秒自动保存
- 数据存储在浏览器本地存储
- 可以手动保存或重置游戏

## 📝 设计理念

参考经典增量游戏（如Cookie Clicker、Universal Paperclips）的核心机制，结合AI大模型行业的真实场景，创造既有趣又有教育意义的游戏体验。

### 核心设计
- **指数增长** - 数值快速膨胀带来的成就感
- **长期目标** - 多层次的解锁和升级系统
- **离线收益** - 支持挂机玩法（待实现）
- **重置机制** - Prestige系统提供元进度（待实现）
