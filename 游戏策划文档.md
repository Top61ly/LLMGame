# 《AI Empire》游戏策划文档

## 📋 文档信息

- **项目名称**：AI Empire - AI大模型公司模拟经营游戏
- **游戏类型**：增量游戏（Idle Game）
- **目标平台**：Web浏览器
- **游戏时长**：长期挂机玩法（可玩50+小时）
- **文档版本**：v1.0
- **更新时间**：2026-03-04

---

## 一、游戏概述

### 1.1 核心概念

玩家经营一家AI大模型公司，通过升级软件、升级硬件、训练数据集来不断提高公司能力。通过发布底层大模型、AI应用、AI硬件来实现盈利，吸引越来越多的用户使用。

**参考游戏**：Cookie Clicker、Universal Paperclips、Trimps

### 1.2 游戏目标

- **短期目标**：
  - 获得第一个CPU集群
  - 累积$10,000
  - 购买10个GPU

- **中期目标**：
  - 解锁所有建筑
  - 达到100级
  - 完成第一次IPO重置

- **长期目标**：
  - 达成终极AGI目标
  - 成为全球顶级AI公司
  - 完成所有156个成就

### 1.3 核心特色

✨ **指数增长** - 数值快速膨胀，带来成就感  
⏰ **长期进度** - 需要50小时+才能完成的海量内容  
💾 **挂机友好** - 离线收益系统，支持纯挂机玩法  
🔄 **重置机制** - Prestige系统提供元进度  
🎯 **多层目标** - 成就、里程碑、研究项目等

## 二、游戏循环

### 2.1 基础循环（第1-2小时）

```
点击训练 → 获得算力 → 用资金买CPU → CPU自动产出 → 升级效率
   ↑____________________________________________系数上升_↓
```

**流程**：
1. 玩家点击"手动训练"按钮 → 获得算力
2. 算力自动转换为资金（汇率可调）
3. 用资金购买硬件（CPU、GPU）
4. 硬件自动每秒产出算力
5. 收益加速，解锁新建筑

### 2.2 进阶循环（第2-10小时）

```
研发升级 → 效率提升 → 产出加倍 → 解锁新模型 → 新收益源
```

**新系统加入**：
- 科技树升级
- 模型训练
- 员工雇佣
- 市场营销

### 2.3 重置循环（10小时+）

```
积累足够资源 → IPO上市 → 获得投资点数 → 永久加成 → 重新开始
```

**Prestige机制**：
- 消耗所有进度获得"投资点数"
- 每个投资点提供0.1%的全局加成
- 支持多次重置，加成可叠加

### 4.2 购买成本公式

```
第n个建筑的成本 = 基础成本 × (成本系数 ^ (n-1))

例如GPU第1个：$500
   GPU第2个：$575（500 × 1.15）
   GPU第3个：$661.25（575 × 1.15）
```

### 4.3 生产能力

```
单位产出 = 基础产出 × 数量 × 里程碑加成 × 研究加成 × 人员加成

里程碑加成：每25个建筑 → 产出×2
研究加成：可通过升级获得 +10% ~ +500%
人员加成：工程师数量 × 0.5% ~ 50%
```

### 4.4 里程碑系统

```
25个      → 50个      → 75个      → 100个
×2产出    ×4产出      ×8产出      ×16产出

【示例】
CPU第1-24个：每秒100
CPU第25个时：触发里程碑，产出翻倍
CPU第25-49个：每秒200
CPU第50个时：触发第二个里程碑
CPU第50+：每秒400
```

### 6.2 模型部署

```
完成训练 → 部署上线 → 获得用户 → 持续收入

【收入计算】
收入/秒 = 模型用户数 × 使用频率 × 调用费率

例如：
DALL-E Mini 10M用户
每个用户每天调用5次
费率 $0.001/次
理论收入 = 10M × 5 × $0.001 / 86400秒 ≈ $0.58/秒
```

### 6.3 模型质量度

```
质量 = 基础 × 训练数据质量 × 算力投入 × 算法等级

高质量模型：
- 用户增长快
- 口碑评分高
- 支持高价格
```

## 十一、Prestige系统（IPO重置）

### 11.1 重置机制

```
触发条件：玩家主动点击"IPO上市"

重置时：
1. 清除所有建筑计数
2. 清除所有资源（金钱、算力等）
3. 清除所有升级进度
4. 清除所有模型部署

保留的东西：
✓ 所有成就历史记录
✓ 投资点数
✓ 永久加成
✓ 游戏时长统计
```

### 11.2 投资点数计算

```
重置时投资点 = √(总资金) / 1000

例如：
总资金 $1 = 0个投资点（不划算）
总资金 $1,000,000 = 31个投资点
总资金 $1,000,000,000 = 1000个投资点
总资金 $1,000,000,000,000 = 31622个投资点
```

### 11.3 永久加成

```
每个投资点提供的加成：
- 起始资金 +0.1%
- 起始算力 +0.1%
- 所有建筑产出 +0.05%
- 所有研究效果 +0.01%

示例（100投资点）：
- 起始资金：$100 × (1 + 10%) = $110
- 起始算力：100 × (1 + 10%) = 110/s
- 建筑产出全部×1.05
```

### 11.4 重置收益评估

```
重置越晚越划算：

第1次重置（2小时）：获得100点 → 下一周期快5倍
第2次重置（3小时）：获得500点 → 下一周期快50倍
第3次重置（4小时）：获得2000点 → 下一周期快500倍

玩家需要权衡：
现在重置 vs 等待更久获得更多加成
```

## 十七、风险与解决方案

| 风险 | 影响 | 解决方案 |
|------|------|---------|
| 数字溢出 | 页面崩溃 | 使用BigInt库或科学计数 |

| 用户流失 | 项目失败 | 定期更新内容 |
| 性能问题 | 卡顿 | 优化渲染+Web Worker |
